## Нужно установить

+ `fastapi`>=0.103.1
+ `jsthon`>=0.4.1

## Установка
```cli
git clone https://github.com/terribleMOTHMAN/iwantincyberzone
cd iwantincyberzone
pip install -r requirements.txt
```
